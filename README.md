Sistema de Gestión Financiera - Juguería Acuario

Este es un sistema de escritorio desarrollado en Python para la automatización, control de ventas, gestión de gastos y seguimiento de clientes deudores (fiados) de la *Juguería Acuario*. El software utiliza una arquitectura modular y persistencia de datos local mediante archivos planos .txt.

Pasos para Ejecutar el Proyecto

Cómo Ejecutar el Programa

Para iniciar el sistema de gestión correctamente, siga estos pasos:

1. Abrir la terminal:** Ingrese a la terminal de comandos (cmd, PowerShell o Git Bash) dentro de la carpeta raíz del proyecto `EF_GRUPOJ`.
2. Ejecutar el archivo principal:** Ejecute el comando correspondiente según su sistema operativo:
   bash
   python main.py
